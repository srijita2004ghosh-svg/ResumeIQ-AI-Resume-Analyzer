# ResumeIQ - AI Powered Resume Analyzer

ResumeIQ is an AI-powered resume analysis tool built using Python and Streamlit.

## Features

* Resume Builder
* ATS Score Calculation
* Skill Gap Analysis
* Resume Strength Evaluation
* Resume Improvement Suggestions
* Resume Keyword Word Cloud
* PDF Resume Export
* Multiple Career Templates

## Technologies Used

* Python
* Streamlit
* Scikit-Learn
* Matplotlib
* WordCloud
* FPDF

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Objective

To help students and job seekers evaluate and optimize their resumes by comparing them against job descriptions and providing actionable feedback.
